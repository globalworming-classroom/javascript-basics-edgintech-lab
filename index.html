At the bottom (line number 36) you should then see code like this: 
    <script type="text/javascript">
    document.getElementById("next").hidden = true
</script>
Within the <script> tag is a Javascript statement that is executed in your browser. We'll talk about that document pretty late in the tutorial. For now it should be enough to realize that this script is hiding the button that leads us to the next page. Change that true to a false and refresh the page in your browser to proceed. 
    <script type="text/javascript">
    document.getElementById("next").hidden = false
</script>
