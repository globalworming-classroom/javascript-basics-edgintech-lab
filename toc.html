This is a very basic overview, there is soooo much to learn. From my experience, these are some topics you need to get started:
  1. Variables
    because we need to memorize things
  2. Types
    because it is useful to distinguish between them
  3. Operators
    because we want to change things
  4. Control Flow
    because we may not want to do things. or do them very often
  5. Arrays
    because everyone needs a list
  6. Objects
    because we want to describe things
  7. Maps
    because they make it easier to build relations
  8. Functions
    because we want reusable scripts and not repeat ourselves
  9. Object Oriented Javascript
    because we want generic and more concrete Objects to model our world
  10. Asynchronous Javascript
    let's do something useful while we wait
  11. Document Object Model (DOM)
    to create elements and interact with the page
  12. Error Handling
    because things might go wrong
