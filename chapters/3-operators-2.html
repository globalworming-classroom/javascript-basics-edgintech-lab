Operators

Logical Operators work on booleans, so true and false values. There are also assignment shorthands for these. 
let stay = true;
const safe = false;
// ||, logical OR
console.log(stay || safe) // -> true
// &&=, logical AND with assignment
stay &&= safe;
console.log(stay) // -> false
// !, logical NOT
console.log(!stay) // -> true
Comparison Operators like <= > == compare values. === is checking for type equality too. 
console.log(1 > 2) // -> false
console.log(2 === 2) // -> true
console.log(2 == "2") // -> true
console.log(2 === "2") // -> false
console.log(10 <= 10) // -> true
Some maybe unexpected results: 
// you can compare alphabetical order
console.log("a" < "b") // -> true
// but all the uppercase letters are "smaller" than lowercase ones
console.log("a" < "A") // -> false
// NaN is not equal to NaN
console.log(NaN == NaN) // -> false
