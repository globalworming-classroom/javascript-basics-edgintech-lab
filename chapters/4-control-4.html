Loops

there are a lot more loops. Like these that are for going through Array entries and Object properties. We have not yet talked about arrays, so let's do that soon :) 
const array = ['a', 1, true];
for (const element of array) {
  console.log(element);
}
// -> 'a'
// -> 1
// -> true

const array = ['a', 1, true];
array.forEach(element => console.log(element))
// -> 'a'
// -> 1
// -> true

const object = { a: 1, b: 2, c: 3 };
for (const property in object) {
  console.log(`${property}: ${object[property]}`);
}
// -> "a: 1"
// -> "b: 2"
// -> "c: 3"
