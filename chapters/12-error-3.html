Finally

The try-catch block can be extended with a finally block. The purpose of that finally is to run regardless of successful or erroneous execution, so you can use it for e.g. cleaning up. 
let accountId
try {
    accountId = startCreatingAccount()
    confirmUserInfo(accountId)
    finishCreation(accountId)
} catch (e) {
    console.error(e)
    throw new ServerError(e)
} finally {
    cleanupAccount(accountId)
}
