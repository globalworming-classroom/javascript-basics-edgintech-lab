Objects

Arrays are objects. Objects have string properties, Arrays are usually just limited to strings of positive integers. 
const array = ["first value"];
array["1"] = "second value"
console.log(typeof array)
// -> object

const object = {"a property": "a value"}
object["another property"] = "another value"
object.thirdProperty = "third value"
console.log(object)
// -> {a property: 'a value', another property: 'another value', thirdProperty: 'third value'}
In comparison to arrays, objects aren't really built for iteration over their properties, you can do it though if need be. Use for-in loop or get an array with key-value pairs 

for (const property in object) {
  console.log(`${property}: ${object[property]}`);
}
// -> a property: a value
// -> anotherProperty: another value
console.log(Object.entries(object))
// -> 0: ['a property', 'a value']
//    1: ['another property', 'another value']
//    2: ['thirdProperty', 'third value']
So, arrays are for keeping track of multiple things, objects are for describing a thing, what it has and does and how to interact with it. This is done by setting properties, string keys. Set values, set other objects or functions. ( We haven't talked about functions yet or what 'this' is. Functions are reusable code blocks, 'this' here referring to the object, so you can access properties) 

const dragon = {
    heads: 1,
    behead: function () {
        this.heads = 0
    }
}
const hydra = {
    heads: 1,
    behead: function () {
        this.heads++
    }
}
