Functions

To make our code more readable (hide details behind a meaningful name), maintainable (don't repeat yourself, structure large programs). To create a callable function, assign a function to a variable 
const randomHero = function () {
    console.log("calling for random hero")
}

or an arrow function to a variable

const randomHero = () => {
    console.log("calling for random hero")
}
The functions body (between the curly braces { }) are the statements that are executed when a function is invoked. Functions may have parameters defined within parentheses ( ) 
const randomHero = function () {
    console.log("calling for random hero")
}
// invoke it by calling
randomHero()
