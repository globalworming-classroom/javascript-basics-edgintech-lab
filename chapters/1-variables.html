Variables

Variables are places where we store values. We declare them with let or const and can set and read variables. 
    const greeting = 'hello';
let name = 'world'
// you can try out and see the logs
// when execution this code in your browsers console
console.log(greeting, name)
// -> hello world
const declares variables that can't be changed. Declared with let, you can reassign them 
    const greeting = 'hello';
greeting = 'hi'
// -> Uncaught TypeError: Assignment to constant variable
    let name = 'Fury';
console.log(name)
// -> Fury
name = 'Peter'
console.log(name)
// -> Peter
use const wherever possible, there rarely is a need to reassign variables and reassigning can be confusing for the reader.
