Arrays

Javascript comes with a lot of useful objects already. We'll focus on Array, Object and Map first. 
const object = Object(); // or const object = {}
const array = Array(); // or const array = []
const map = new Map()
These can look and feel quite similar because you could think of them as containers for key-value pairs 
const object = { 0: "first value" }
const array = [ "first value" ]
const map = new Map([[0, "first value"]]);

object[0] // -> 'first value'
array[0] // -> 'first value'
map.get(0) // -> 'first value'
Let's start with arrays because they are arguably the simplest. Arrays are structure to hold many things, so you can use them as lists, stacks or queues. You can construct arrays in multiple ways. Adding items will automatically increase the Arrays length. 
// initializer notation
const array1 = ["a", "b"]
// Array with or without new
const array2 = new Array("a", "b")
const array3 = Array("a", "b")
// empty Array with defined length
const array4 = Array(4)
console.log(array4)
// -> [,,,]
const array5 = []
console.log(array5.length)
// -> 0
array5[0] = "hi"
console.log(array5.length)
// -> 1
Array values don't have to be of the same type, though there seldom is a reason to do so. Arrays are usually for keeping a list of similar things. 
const array = ["yes", true, 1];
for (const value of array) {
    console.log(typeof value)
}
// -> string, boolean, number
But sometimes Arrays hold different things 
const entry = ["id", 244882];
const key = entry[0]
const value = entry[1]
