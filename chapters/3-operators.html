Operators

Operators invoke special functions. Like typeof operator resulting in the type of that value being returned. There are many operators, like assignment operators like = += ... 
let i = 1
i *= 5 // same as i = i * 5
console.log(i) // -> 5
arithmetic operators like + - / * % ++ ... 
let i = 1
console.log(i) // -> 1
console.log(++i) // -> 2
console.log(++i / 2) // -> 1.5



const parsedFloat = parseFloat("11.5")
console.log(parsedFloat, typeof parsedFloat) // -> 11.5, 'number'
const parsedInt = parseInt("11.5")
console.log(parsedInt, typeof parsedInt) // -> 11, 'number'
