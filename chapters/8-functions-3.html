Spread Operator

Let's look a bit more at the spread operator because it is so useful: it spreads any iterables, making each entry a function parameter, so converting iterables to arrays is super easy 
// create keys and values arrays
const map = new Map([["thunder", "lightning"],["fire","ice"]]);
const keys = [...map.keys()]
console.log(keys)
const values = [...map.values()]
console.log(values)

// list of unique entries by using new Set()
const entries = ["apple", "banana", "grape", "banana"]
const uniqEntries = [...new Set(entries)]
console.log(uniqEntries)

// array of DOM elements
const errorMessageNodes = [...document.getElementsByClassName("error")]
console.log(errorMessageNodes)

// characters in a string
console.log([..."hello"])
Concatenate Arrays 
const lat = ["N", "S"]
const long = ["E", "W"]
const directions = [...lat, ...long]
console.log(directions)
// -> [ 'N', 'S', 'E', 'W' ]
Copy Arrays 
const source = [{"number" : 1}, {"number" : 2}, {"number" : 3}]
const copy = [...source]
console.log(copy)

source[0].number += 1 // also changes entry of copy
console.log(copy)

source[0] = undefined // but changing one array doesn't change the other
console.log(source)
console.log(copy)
