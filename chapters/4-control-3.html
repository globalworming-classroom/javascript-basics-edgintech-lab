Loops

For loops have multiple components. After the for keyword, in the parentheses, we start with an initialization block followed by a condition check. If the condition resolves to true, the inner block is executed and then the last statement in the parentheses is executed. 
for (
    // 1. init
    let counter = 0;
    // 2. check condition
    counter <= 6;
    // 4. execute after inner code block
    counter++
) {
    // 3. code block
    console.log(counter);
}
// -> 0, 1, ..., 6
We did all that with while-loops too, it just looked a bit different. 
// 1. init
let counter = 0
while (
    // 2. check condition
    counter <= 6
) {
    // 3. code block
    console.log(counter);
    // 4. execute at the end
    counter++
}
All the blocks in the parentheses of a for loop are actually optional, you can use some or neither of these. 
// endless loop
for (;;) {
    console.log("executing block");
}
There are fancy things you can do but normally don't, like multiple variables involved, more complex conditions and after loop statements... 

for (let i = 3, j = 0; i < 10; i += 6, j++) {
  console.log(i, j);
}
// -> 3 0
// -> 9 1
