Conditional

You sometimes want to do different things. The if keyword with a condition in parentheses is used to create execution branches. The statement is only executed when the condition evaluates to true. 
if (7 < 13) console.log("7 is less than 13")
Often we need to distinguish between an expression being true or false, the "else" statement is used here 
if (Math.random() < 0.5) console.log("random number less 0.5")
else console.log("greater or equal 0.5")
Multiple statements need to go in a block (and it's usually a good format to put single statements into curly brakets as well) 
if (Math.random() < 0.5) {
    console.log("true")
    console.log("random number less than 0.5")
} else {
    console.log("greater or equal 0.5")
}
If you want to distinguish more conditions, the else-if blocks allow that. The inner block of the first condition that is true is executed. 
const randomNumber = Math.random()

if (randomNumber < 0.5) {
    console.log("random number less than 0.5")
} else if (randomNumber < 0.9) {
    console.log("less than 0.9 but bigger or equal to 0.5")
} else {
    console.log("greater or equal 0.9")
}
Checking for undefined and null can be replaced with the optional chaining operator ?. so property access will return undefined instead of throwing an error. 
const array = ["text"]
let result;
if (array[0]) result = array[0].toUpperCase()
// same as
const array = ["text"]
let result = array[0]?.toUpperCase()
