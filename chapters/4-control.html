Loops

Repeat things a lot, with variants how. While loops are the simplest, just repeat something as long as a provided condition is true 
while (true) { // endless loop, don't execute :)
    console.log("say hi")
}
The condition is checked first and if true, the inner block is executed. Execution stops when courses reaches 10 
let courses = 0
while (courses < 10) {
    console.log("subscribe")
    courses++
}
You can exit loops early with the break keyword 

let courses = 0
while (true) {
    courses++
    break
}
console.log(courses) // -> 1
