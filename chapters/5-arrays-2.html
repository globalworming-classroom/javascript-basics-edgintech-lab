Arrays

Arrays hold values. The keys (or properties) to get these values are the string representations of integers greater or equal zero. 

const array = ["battery", "horse", "staple"];
console.log(array[0])
// -> "battery"
console.log(array[2])
// -> "staple"
// or by string representation of number
console.log(array["1"])
// -> "horse"
Fun fact: you can set other properties as well. You shouldn't though :) 

const array = ["battery", "horse", "staple"];
array[-1] = "minusOne"
array["someProperty"] = "someValue"
You can loop through Array values in many ways, the more common ones are 
// for loop with index based access
const array = ["a", "b", "c"]
for (let i = 0; i < array.length; i++) {
    console.log(`${i}: ${array[i]}`)
}


// for-of loop
const array = ["a", "b", "c"]
for (const value of array) {
    console.log(value)
}
or if you want to make it especially hard for yourself, use the array iterator 
const array = ["a", "b", "c"]
const iterator = array.keys()
let next = iterator.next();
while (!next.done) {
    console.log(`${next.value}: ${array[next.value]}`)
    next = iterator.next()
}
